Строка запуска:
java -jar javaRabbitMessageReceiver-1.0.jar HOST LOGIN PASSWORD ЗАДЕРЖКА_МЕЖДУ_ОБРАБОТКОЙ_СООБЩЕНИЙ_МИЛЛИСЕКУНДЫ

Важные моменты:
в папке, из которой выполняется запуск, должны присутствовать следующие файлы/папки:
lib - папка с бибилиотеками
properties - папка, которая содержит файл log4j.properties
javaRabbitMessageReceiver-1.0.jar - джарка запуска
